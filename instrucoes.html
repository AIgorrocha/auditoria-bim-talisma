3. Como Verificar um Item (Passo a Passo)

Para o auditor verificar qualquer item da auditoria:

1
Abra o Relatório de Auditoria
Abra o arquivo RELATORIO AUDITORIA BIM - TALISMA.html no navegador (ou acesse online pelo link).
2
Navegue até o item
Escolha a edificação na aba superior → Clique na disciplina (ex: ARQUITETURA) → Clique na categoria (ex: COBERTURA) → Encontre o item pelo código.
3
Verifique a referência
Cada grupo de itens mostra:
  • Nome da Tabela — tabela de quantitativos do Revit de onde veio o dado
  • Nome do Arquivo — modelo BIM utilizado
  • 📄 Abrir XLSX — clique para abrir a planilha de quantitativos
  • 🧊 Abrir RVT — clique para abrir o modelo no Revit
4
Confira no arquivo XLSX
Ao abrir o XLSX, procure pela tabela indicada no relatório. A quantidade extraída do modelo estará lá. Compare com a coluna "Qtd BIM" do relatório.
5
(Opcional) Confira no modelo Revit
Se quiser ir mais a fundo, abra o arquivo .rvt no Autodesk Revit. Navegue até a tabela (View → Schedules) com o nome indicado no relatório para verificar a extração diretamente do modelo 3D.
Importante: Os links "Abrir XLSX" e "Abrir RVT" só funcionam quando o HTML é aberto de dentro desta pasta no computador local. Se acessar online pela Vercel, use os arquivos XLSX que estão nas subpastas de cada edificação.

4. Descrição dos Dashboards

Dashboard Completo

Visão geral da auditoria com aditivos E retroativos:

  • Painel Geral — KPIs (valor do orçamento, aditivo, retroativo, itens auditados), gráficos por edificação e disciplinas
  • Por Edificação — Tabela comparativa das 5 edificações + tabela geral das 10
  • Top Divergências — 30 itens com maior impacto financeiro
  • Por Categoria — Agrupamento por tipo de serviço
  • Metodologia — Escopo, processo, limitações

Dashboard Aditivos

Versão focada apenas nos itens de aditivo (BIM > Orçamento):

  • 131 itens onde o levantamento BIM encontrou quantidade superior ao orçamento
  • Valor total dos aditivos: R$ 2.236.048,72
  • Inclui lista completa dos 131 itens com busca/filtro
  • Não inclui itens retroativos (onde o orçamento era maior que o BIM)

Relatório de Auditoria

Documento técnico de rastreabilidade para o auditor:

  • 1.268 itens organizados por Edificação → Disciplina → Categoria → Tabela BIM
  • Links diretos para os arquivos XLSX e RVT
  • Justificativa/método de verificação para cada item
  • Aba de rastreabilidade com todos os 41 arquivos BIM e suas tabelas

7. Requisitos Técnicos

  • Para visualizar dashboards e relatório: Qualquer navegador moderno (Chrome, Edge, Firefox)
  • Para abrir arquivos XLSX: Microsoft Excel ou Google Sheets
  • Para abrir arquivos RVT: Autodesk Revit 2024 ou superior
  • Acesso online: Conexão com internet (para carregar os gráficos via Chart.js)
Sobre os links de arquivos no relatório: Os botões "Abrir XLSX" e "Abrir RVT" dentro do Relatório de Auditoria funcionam apenas quando o arquivo HTML está na mesma pasta que as subpastas das edificações. Mantenha a estrutura de pastas original.
